# minicord

A tiny Discord API library for freethreaded Python.

It is not intended to be featureful, easy to use, or particularly well written. It is fast, though!
And also, it's somewhat similar to Pycord.

## Notes on free-threading

- Free-threaded Python is still evolving; package compatibility varies.
- This library itself is pure Python and intended to be simple to experiment with under `3.13t`.
